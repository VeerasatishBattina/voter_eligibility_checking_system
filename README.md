# voter_eligibility_checking_system
vote is like a weapon in society to elect right person to develop the society.
The Voter Eligibility Checking System is a simple web application that checks whether a person is eligible to vote based on the age criterion. It helps quickly determine voter eligibility by verifying if the user meets the minimum voting age requirement
Features
Checks voter eligibility based on age
Simple and easy input system (age entry)
Instant result display (Eligible / Not Eligible)
Lightweight and beginner-friendly project
Responsive and easy-to-use interface
The system takes the user’s age as input and compares it with the minimum voting age requirement (usually 18 years).
If age ≥ 18 → Eligible to Vote
If age < 18 → Not Eligible to Vote
This project demonstrates a basic rule-based decision-making system using a real-world use case of voter eligibility based solely on age
